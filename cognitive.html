Sensory Memory

Information first enters through sensory registers:

  • Iconic (visual) memory: ~250ms duration
  • Echoic (auditory) memory: 2-4 seconds
  • Haptic (touch) memory

Working Memory

Active processing of information:

  • Phonological loop for verbal content
  • Visuospatial sketchpad for visual information
  • Central executive for coordination

Long-term Memory

Permanent storage system:

  • Declarative memory (facts and events)
  • Procedural memory (skills and procedures)
  • Episodic memory (personal experiences)
